### 🌟 Flagship Project

#### [🛡️ Safeguard AI: Clinical-Grade Multimodal Triage System](https://github.com/MohitParmar78/MindGuard-AI-Mental-Health-System)
A production-ready, hallucination-free Machine Learning architecture designed to provide verifiable and empathetic mental health triage. 
* **Custom ML Triage:** Built a custom PyTorch trainer to fine-tune an XLM-RoBERTa classification engine, overriding standard loss mechanics for severe class imbalance across 35 distinct emotional states.
* **Factual Grounding (RAG):** Architected a ChromaDB vector store that intercepts the diagnosis to fetch medically verified coping strategies, strictly grounding the generation model (Llama-3.3-70B via Groq API).
* **Explainable AI (XAI):** Integrated SHAP (Game Theory) to calculate token-level feature importance during runtime, mathematically proving exactly which words triggered the neural network's assessment.
* **MLOps & Multimodal UI:** Bypassed cloud OOM limits by abstracting 1.1GB model weights to a remote registry. Engineered persistent Streamlit state-management for real-time OpenAI Whisper audio transcription and live mood tracking.
